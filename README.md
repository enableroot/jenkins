# Jenkins

Bu repo, Docker ile tek bir `Dockerfile` kullanarak Jenkins çalıştırmak için hazırlanmıştır.

**Proje:**
- **Amaç:** Tek Dockerfile ile Jenkins imajı oluşturmak.


**Admin Parolası**
- **Yol:** Başlangıçtaki admin parolası container içinde `/var/jenkins_home/secrets/initialAdminPassword` dosyasında bulunur.
